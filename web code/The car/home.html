DIY Arduino Gesture Control Robot.

welcome To Azza

Presenting our remarkable robot car project featuring dual modes for enhanced functionality. In autonomous mode, the car adeptly navigates while avoiding barriers using flame and ultrasonic sensors. Switching to manual mode, intuitive hand gestures through an ADXL accelerometer control the car's movements. Both modes integrate vital sensor inputs for precise decision-making. Accompanying this innovation, a dynamic Flutter application provides real-time sensor data display and remote control capabilities. Additionally, a web application mirrors these functions, ensuring accessibility. This project seamlessly integrates robotics, sensor technology, gesture control, and multi-platform applications, exemplifying our commitment to cutting-edge, interactive solutions.

It's time to try our beautiful car!

How does it work?

  • A buzzer and a red led will activate when the distance is too close, and the buzzer and red led will also function when the robotic car is moving with the aid of a controller composed of an LCD and keyboard.

  • The robotic car has a flame sensor and an ultrasonic sensor to detect the presence of flame.

  • We developed a web and flutter application for it, and after that, we transferred the data from the sensor readings to Google's firebase service.
